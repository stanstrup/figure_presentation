# Claude Code Instructions

## Plot Output Conventions

All generated plots should be saved to the `plots/` folder in topic-specific subfolders:

```
plots/
├── 01_formats/
├── 03_formats/
├── 04_sizing/
├── 06_saving/
├── 07_editing/
├── 09_powerpoint/
├── 10_rendering/
└── 11_interactive/
```

### Naming Convention
- Use the topic number prefix matching the qmd file (e.g., `04_sizing/` for `04_sizing.qmd`)
- Use descriptive filenames: `sizing_small.svg`, `vector_demo.svg`, etc.

### In QMD Files
- ggsave paths: `ggsave("plots/04_sizing/filename.svg", ...)`
- Markdown image refs: `![](plots/04_sizing/filename.svg){width="100%"}`

### Static Assets
The `sources/` folder is for static assets (images, external files) NOT generated by R code.
